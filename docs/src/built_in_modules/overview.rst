=========================
Built-in Modules Overview
=========================

This section collects Rogue's built-in, batteries-included capabilities.
It groups production modules and integrations that are commonly used directly
in applications.

Utilities
=========

This section describes the various utilities available in Rogue to support
file operations, PRBS transmit and receive as well as stream compression.

- :doc:`/utilities/index`

Hardware
========

This section describes the hardware which is supported by Rogue with the
help of a Linux kernel device driver. The device driver required to use
these interfaces can be found at:

https://github.com/slaclab/aes-stream-drivers

- :doc:`/hardware/index`
