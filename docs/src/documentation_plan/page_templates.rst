==============
Page Templates
==============

Use these lightweight templates to keep section intent consistent.

Tutorial Template
=================

- Goal and expected outcome
- Prerequisites
- Step-by-step procedure
- Validation checks
- Links to related API reference

How-to Template
===============

- Task statement
- Inputs and prerequisites
- Procedure
- Troubleshooting
- Links to related conceptual pages and API reference

Explanation Template
====================

- Problem/model framing
- Core concepts and relationships
- Design tradeoffs and constraints
- Links to concrete how-to pages
- Links to API reference

Reference Template
==================

- Type/class/function signatures
- Parameters/returns
- Side effects/errors
- Minimal behavior notes
- Link to at least one conceptual page
