=======================
First Success Path
=======================

Goal
====

Get a minimal PyRogue tree running and reachable from a client in the shortest
possible path.

Prerequisites
=============

- Rogue installed: :doc:`/installing/index`
- Python environment with ``pyrogue`` import available

Steps
=====

1. Build a minimal root/device tree using:
   :doc:`/getting_started/pyrogue_tree_example/p_t_root`
2. Add a basic custom device using:
   :doc:`/getting_started/pyrogue_tree_example/p_t_device`
3. Validate your final structure against:
   :doc:`/getting_started/pyrogue_tree_example/full_root`
4. Verify client access basics:
   :doc:`/interfaces/clients/simple`

Validation checklist
====================

- Root starts and stops cleanly.
- You can read/write at least one variable.
- Client can connect to the tree and issue a basic command.

Next step
=========

Proceed to :doc:`/tutorials/index` for deeper workflow-oriented learning.
