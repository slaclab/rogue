=================
Example Pipelines
=================

Pipeline examples for sending, receiving, and connecting stream modules:

Recommended progression
=======================

1. Start with a minimal producer/consumer path.
2. Add one processing stage and validate correctness.
3. Add buffering and rate-control where throughput mismatch appears.
4. Add instrumentation for sustained-rate validation.

Use this page as a recipe gateway. For deeper guidance, pair these examples
with :doc:`/stream_interface/overview` and
:doc:`/stream_interface/built_in_modules`.
