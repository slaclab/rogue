PyRogue Tree Getting Started Guide

This guide provides a quick walkthrough of a basic PyRogue tree application. It walks from a minimal root setup through custom device definition, then finishes with complete reference code.

Recommended Learning Path

  1. Build Your First PyRogue Root - build a minimal working root and add common components.

  2. Create A Custom PyRogue Device - define a custom pyrogue.Device with RemoteVariable, LinkVariable, and RemoteCommand entries.

  3. Review The Complete Root Example - review the full assembled example in one place.

  4. PyRogue Tree - review the PyRogue Tree architecture and concepts.

  5. Stream Interface and Memory Interface - review lower-level interfaces as needed.
