class pyrogue.LinkVariable(*, name, variable=None, dependencies=None, linkedSet=None, linkedGet=None, minimum=None, maximum=None, **kwargs)[source]

Variable linked to another variable or custom functions.

Parameters:

  • name (str) – Variable name.

  • variable (BaseVariable | None, default: None) – Optional variable to link to. This is a useful shortcut for linking to a variable without having to declare dependencies.

  • dependencies (Iterable[BaseVariable] | None, default: None) – List of variables that this variable depends on.

  • linkedSet (Callable[[Any, BaseVariable, Any, bool, int, bool, bool], Any] | None, default: None) – Setter callback. Expected keyword arguments are dev, var, value, write, index, verify, check. These match the parameters of BaseVariable.set(). The callback may accept any subset of these names. Only value is required. If not provided, will infer mode = ‘RO’.

  • linkedGet (Callable[[Any, BaseVariable, bool, int, bool], Any] | None, default: None) – Getter callback. Expected keyword arguments are dev, var, read, index, check. These match the parameters of BaseVariable.get(). The callback may accept any subset of these names.

  • minimum (Any | None, default: None) – Minimum allowed value.

  • maximum (Any | None, default: None) – Maximum allowed value.

  • **kwargs (Any) – Additional arguments forwarded to BaseVariable.
