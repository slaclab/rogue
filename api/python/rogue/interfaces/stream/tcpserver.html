C++: rogue::interfaces::stream::TcpServer

Binding source: [TcpServer.cpp]

Stream TCP bridge server.

Thin wrapper around TcpCore configured for server mode.

__init__(addr, port)

C++: rogue::interfaces::stream::TcpServer

Constructs a TCP stream bridge server.

This constructor is a low-level C++ allocation path. Prefer create() when shared ownership or Python exposure is required. The address can be an IP address or hostname and selects the local bind interface. A value of "*" binds all interfaces. The bridge uses two consecutive TCP ports; port is the base (for example, port=8000 uses 8000 and 8001).

Parameters:

  • addr – Local bind address.

  • port – Base TCP port number.

close()

C++: rogue::interfaces::stream::TcpCore::close()

Closes active bridge connections.
